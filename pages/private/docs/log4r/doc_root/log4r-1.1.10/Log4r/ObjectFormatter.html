    • #format

    class Log4r::ObjectFormatter

    Formats objects the same way irb does:

    loggername:foo.rb in 12> 
[1, 3, 4]
loggername:foo.rb in 13> 
{1=>"1"}

    Strings don't get inspected. just printed. The trace is optional.
