|Property | Value |Default |Description

|`flex-flow`
|`flex-direction` `flex-wrap`
|`row nowrap`
|The `flex-flow` property is a *shorthand* propery for setting both the
`flex-direction` and `flex-wrap` properties.

.Example
[source, css]
----
.flex-container {
  display: flex;
  flex-flow: row wrap;
}
----
